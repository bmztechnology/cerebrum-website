# Order Block: Institutional Footprints

## I. Definition
Last bullish/bearish candle before strong move.

## II. Bullish Order Block
Last DOWN candle before UP impulse.
= Institutional buying zone.

## III. Bearish Order Block
Last UP candle before DOWN impulse.
= Institutional selling zone.

## IV. Trading
When price returns to order block:
- Bullish OB = Buy opportunity
- Bearish OB = Sell opportunity

## V. Validity
- Must have strong impulse after
- Fresh (untested) = Stronger
- Higher TF = More significant

## VI. Stop Placement
Beyond order block's opposite side.

## VII. vs Support/Resistance
OB is more specific: the exact candle of institutional activity.
